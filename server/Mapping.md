# Packet Mapping

## ``GameState``

Represents the current state of the game.

| Field         | Type        | Description                     |
|---------------|-------------|---------------------------------|
| `0`			| `OFF`       | Game is off						|
| `1`			| `REPLAY`    | Game is showing a replay 		|
| `2`			| `LIVE`      | Game is live					|
| `3`			| `PAUSED`    | Game is paused					|
